1這兩個工具是什麼？

工具角色一句話介紹
Wave Terminal 終端機（容器） 開源、跨平台的現代終端機，內建檔案預覽、網頁瀏覽、AI 助手等圖形化功能
Claude Code AI 編程助手（住在終端機裡） Anthropic 官方 CLI，在終端機裡用對話方式寫程式、改檔案、跑指令

兩者關係：Wave Terminal 是「房子」，Claude Code 是「住客」。先裝哪個都可以，但建議先裝 Wave Terminal，之後所有指令都在它裡面執行，順便熟悉操作。

2安裝 Wave Terminal

官方下載頁：https://www.waveterm.dev/download

Windows

方法 A — 直接下載安裝檔（最簡單）：

  • EXE 安裝程式（x64）：Wave-win32-x64-0.14.5.exe
  • MSI 安裝程式（x64）：Wave-win32-x64-0.14.5.msi

下載後雙擊執行，一路下一步即可。

方法 B — 用 winget（Windows 內建套件管理器），打開 PowerShell 輸入：

winget install CommandLine.Wave

macOS

方法 A — 直接下載 DMG：

  • Apple Silicon（M1～M4）：Wave-darwin-arm64-0.14.5.dmg
  • Intel 晶片：Wave-darwin-x64-0.14.5.dmg

方法 B — 用 Homebrew：

brew install --cask wave

Linux

發行版格式下載 / 指令
Ubuntu / Debian.debwaveterm-linux-amd64-0.14.5.deb
Fedora / CentOS.rpmwaveterm-linux-x86_64-0.14.5.rpm
各發行版通用Snapsnapcraft.io/waveterm（snap install waveterm）
Arch Linux.pacmanwaveterm-linux-x64-0.14.5.pacman
可攜式AppImage在官方下載頁選 AppImage（x64 / ARM64）
ℹ️ 版本號說明
上面連結是查證當下（2026-06-05）的最新版 0.14.5。日後版本更新時，直接到 官方下載頁 抓最新版即可，頁面永遠指向最新版本。

裝好後打開 Wave Terminal，會看到分頁式介面＋側邊的圖形化區塊（widget）。它本質仍是終端機——接下來的所有指令都在這裡輸入。

3安裝 Claude Code

官方安裝文件：https://code.claude.com/docs/en/setup

系統需求

  • 作業系統：macOS 13.0+、Windows 10 1809+ / Windows Server 2019+、Ubuntu 20.04+ / Debian 10+ / Alpine 3.19+
  • 硬體：4 GB+ RAM、x64 或 ARM64 處理器
  • 網路：需要網際網路連線
  • 帳號：Claude Pro / Max / Team / Enterprise 或 Console（API）帳號 —— 免費版 Claude.ai 不能用 Claude Code

安裝方式（擇一即可）

Windows　官方推薦的原生安裝器（Native Install），打開 PowerShell（或 Wave Terminal）輸入：

irm https://claude.ai/install.ps1 | iex

如果你用的是 CMD（命令提示字元），改用這行：

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
⚠️ 怎麼分辨自己在 PowerShell 還是 CMD？
提示符開頭有 PS C:\ 就是 PowerShell；只有 C:\ 就是 CMD。
・在 PowerShell 跑 CMD 的指令會出現 The token '&&' is not a valid statement separator
・在 CMD 跑 PowerShell 的指令會出現 'irm' is not recognized
看到這兩種錯誤，代表你用錯指令了，換另一行即可。

macOS Linux / WSL　原生安裝器：

curl -fsSL https://claude.ai/install.sh | bash

其他安裝管道：

管道指令備註
winget（Windows）winget install Anthropic.ClaudeCode不會自動更新，需手動 winget upgrade
Homebrew（macOS）brew install --cask claude-code不會自動更新，需手動 brew upgrade
npm（需 Node.js 18+）npm install -g @anthropic-ai/claude-code切勿加 sudo
✅ 推薦用原生安裝器
原生安裝（install.ps1 / install.sh）會自動在背景更新，裝一次就不用管了。winget / brew / npm 裝的都要自己手動更新。
ℹ️ Windows 使用者建議加裝 Git for Windows
官方建議在原生 Windows 上安裝 Git for Windows，這樣 Claude Code 可以使用 Bash 工具（功能較完整）；沒裝的話會改用 PowerShell 執行指令，也能正常運作。不需要管理員權限。

驗證安裝

claude --version

有顯示版本號就成功了。想做更完整的健檢可以跑：

claude doctor

4第一次啟動與登入

1
切到你的專案資料夾（或任何想工作的資料夾）： 
cd C:\你的專案資料夾
2
輸入 claude 啟動： 
claude
3
跟著畫面提示登入 —— 會自動打開瀏覽器，用你的 Claude 帳號（Pro / Max / Team）或 Console 帳號授權，完成後回到終端機就能開始對話。
4
開始使用 —— 直接用自然語言下指令，例如「幫我看一下這個資料夾的結構」「寫一個 hello world 的 Python 程式」。

5合體：在 Wave Terminal 裡跑 Claude Code

兩個都裝好後，組合方式非常簡單：

  1. 打開 Wave Terminal
  2. 在終端機分頁輸入 claude
  3. 完成 —— 你現在擁有「圖形化終端機 + AI 編程助手」的組合

這個組合的實際好處：

  • 分頁工作流：一個分頁跑 Claude Code，另一個分頁自己下指令、看 log
  • 檔案預覽 widget：Claude Code 改完的檔案、生成的圖片，直接在 Wave 側邊預覽，不用切視窗
  • 內建瀏覽器 widget：查文件、看 Claude Code 生成的 HTML 都在同一個視窗內
✅ 中文顯示亂碼？
Wave Terminal 預設字型對 CJK（中日韓）支援不佳，中文可能變豆腐字。解法：到設定檔 ~/.config/waveterm/settings.json 把字型改成支援中文的等寬字型（例如 Sarasa Mono TC 更紗黑體），設定 "term:fontfamily": "Sarasa Mono TC" 後重啟即可。

6常見問題排除

症狀原因 / 解法
claude : 無法辨識 / command not found 安裝後 PATH 還沒更新 → 關掉終端機重開一次再試。還是不行就參考官方 安裝疑難排解。
登入時說帳號不能用 Claude Code 需要 Pro / Max / Team / Enterprise / Console 帳號，免費版不支援。先到 claude.ai 升級方案。
'irm' is not recognized 你在 CMD 裡跑了 PowerShell 指令 → 改用 CMD 版安裝指令（見上方第 3 節）。
The token '&&' is not a valid... 你在 PowerShell 裡跑了 CMD 指令 → 改用 irm https://claude.ai/install.ps1 | iex。
npm 安裝時權限錯誤 不要用 sudo npm install -g（官方明確警告）。建議改用原生安裝器。
想手動立即更新 Claude Code 跑 claude update；原生安裝平常會自動背景更新。
不想用終端機 Claude Code 也有桌面版 App（圖形介面），以及 VS Code / JetBrains 外掛可選。
