## 這個資料夾怎麼用

- **要公開的 `.html` 檔**：直接丟進這裡（或子資料夾）
- **更新首頁連結清單**：編輯 `index.html` 裡的 `<ul>` 區塊
- **上線**：雙擊 `發佈.bat`（全自動 commit + push）
- **私密資料**：請放 `C:\Users\whatp\做好的資料\` 根目錄，不要進來這個資料夾

## 禁忌清單（別推上 GitHub）

- 任何含金額 / 物件地號 / 客戶姓名的檔案
- `.env` / API key / 密碼
- 民宿財務、稽查報告、物件合約
